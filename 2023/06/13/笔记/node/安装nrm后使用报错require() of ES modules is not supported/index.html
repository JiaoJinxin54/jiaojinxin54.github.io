安装nrm后使用报错require() of ES modules is not supported

发表于2023-06-13|更新于2024-03-06|CentOSnrm
|字数总计:340|阅读时长:1分钟

在CentOS 7系统、node v14.21.3环境下使用npm安装nrm后，报错如下：

应该使用open的CommonJs规范包，现在open v9.0.0是ESModule版本的包。

解决方案：

执行npm uninstall nrm卸载当前nrm依赖包；

执行npm install -g nrm open@8.4.2 --save重新安装新的nrm依赖包。
